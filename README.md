# Gincraft

A CLI tool for quickly scaffolding Gin web applications with a clean architecture structure.

## Installation

```bash
go install github.com/otobongfp/gincraft@latest
```

## Usage

Create a new Gin project:

```bash
gincraft new myapp
cd myapp
go mod tidy
go run main.go
```
